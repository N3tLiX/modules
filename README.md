# modules
Collection of Azure (azurerm) Terraform modules
## Usage in Terraform 0.13
```hcl
module "example" {
  source  = "github.com/N3tLiX/modules//[MODULE_FOLER_NAME]"
}
```

Find an example, and more documentation at https://github.com/n3tlix/examples
